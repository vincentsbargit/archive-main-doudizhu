# archive-main-doudizhu

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，便于浏览和长期维护。

## 项目简介

`archive-main-doudizhu` 是一个专注于 HTML 页面归档的仓库。所有页面均为独立文件，不依赖特定后端服务或数据库。仓库主要面向对页面内容存档、历史版本回溯有需求的场景。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库主页索引（可选）
├── pages/             # 归档页面存放目录
│   ├── example1.html
│   └── example2.html
└── assets/            # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有归档的 HTML 页面，每个文件为一个独立页面。
- `assets/`：存放页面引用的公共资源文件，按类型分类。

## 页面归档说明

- 所有页面均为静态 HTML，不包含外部依赖（除非在 `assets/` 中明确提供）。
- 页面文件名建议使用有意义的英文或拼音命名，避免特殊字符。
- 如需更新页面内容，请保留原文件并创建新版本，或直接覆盖并更新提交说明。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，或修正已有页面中的错误。
- 提交时请附上简要说明，方便其他维护者了解变更内容。
- 本仓库不提供任何形式的商业服务或技术支持，仅作为页面存档使用。
- 请勿提交包含恶意代码、广告或违反法律法规的内容。
